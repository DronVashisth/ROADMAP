Beginner Coding Journey Roadmap

Follow this roadmap to start your coding journey step by step. Click each step to see daily details and focus areas!

Step 1: Learn HTML (Days 1-15)

Day 1: Introduction to HTML, basic structure, and tags like <html>, <head>, and <body>.

Day 2: Learn headings, paragraphs, and text formatting tags.

Day 3: Explore links and anchor tags.

Day 4: Work with images and attributes.

Day 5: Create lists (ordered and unordered).

Day 6: Learn tables and table attributes.

Day 7: Forms: input, textarea, button, and labels.

Day 8: Forms: dropdowns, checkboxes, and radio buttons.

Day 9: Semantic tags like <header>, <footer>, and <article>.

Day 10: Inline vs block elements.

Day 11: Create a simple webpage structure.

Day 12: Add navigation menus.

Day 13: Explore meta tags and their importance.

Day 14: Optimize HTML for SEO basics.

Day 15: Build a complete static webpage project.

Resources: W3Schools, YouTube tutorials.

Step 2: Learn CSS (Days 16-30)

Day 16: Introduction to CSS and its syntax.

Day 17: Learn selectors, properties, and values.

Day 18: Understand the box model.

Day 19: Work with colors, backgrounds, and borders.

Day 20: Learn text styling and alignment.

Day 21: Introduction to Flexbox.

Day 22: Advanced Flexbox concepts.

Day 23: Learn Grid layout basics.

Day 24: Advanced Grid layout.

Day 25: Responsive design basics (media queries).

Day 26: Create a responsive webpage layout.

Day 27: Animations and transitions.

Day 28: Learn pseudo-classes and pseudo-elements.

Day 29: Review and refine CSS skills.

Day 30: Build a responsive portfolio project.

Resources: CSS Tricks, YouTube tutorials.

Step 3: Learn Bootstrap Framework (Days 31-40)

Day 31: Introduction to Bootstrap and setup.

Day 32: Learn the grid system.

Day 33: Explore utility classes.

Day 34: Work with buttons and forms.

Day 35: Learn navigation bars and menus.

Day 36: Use modals and cards.

Day 37: Build a simple responsive webpage.

Day 38: Review Bootstrap components.

Day 39: Add animations and interactivity.

Day 40: Build a complete project using Bootstrap.

Resources: Bootstrap official docs, YouTube tutorials.

Step 4: Learn JavaScript (Days 41-60)

Days 41-45: Learn variables, data types, and operators.

Days 46-50: Work on loops and conditional statements.

Days 51-55: Learn functions, arrays, and objects.

Days 56-60: Build a small interactive project like a to-do app.

Resources: MDN Web Docs, YouTube tutorials.

Step 5: Learn PHP & MySQL (Days 61-90)

Days 61-75: Learn PHP basics and how to handle forms.

Days 76-85: Connect your PHP code to a MySQL database.

Days 86-90: Build a simple backend project (e.g., a blog or login system).

Resources: PHP.net, MySQL official docs.

Step 6: Learn Node.js & React.js (Days 91-120)

Days 91-105: Master Node.js basics and build a simple API.

Days 106-120: Learn React basics and build a dynamic front-end project.

Resources: Node.js official docs, React official docs.
